UED 素质扩展之趣味乒乓球比赛

发表于 2018-05-08 | 更新于: 2018-06-04 | 分类于 运动 | 阅读次数: 字数统计: 994 | 阅读时长 ≈ 3
为了丰富大家的文化生活，培养大家团队意识，为大家的生活、工作带来乐趣，特举行乒乓球趣味活动，希望大家踊跃参加。

活动主题：素质扩展之趣味乒乓球比赛;

活动目的：为了丰富大家的文化生活，培养大家团队意识，为大家的生活、工作带来乐趣，特举行乒乓球趣味活动，希望大家踊跃参加;

活动形式：首先进行乒乓球趣味赛，其次进行乒乓球团体赛;

活动时间：2018-5-10 ~ 2018-5-31;

活动地点：研发大楼14楼，乒乓球室;

活动流程：

主持及裁判人：韩森、徐雅茹 ** 暂定 （共三位、裁判也是参赛选手）

乒乓球趣味赛：根据团队人员共分为4个组，每组5人，每次进行一个项目趣味比赛，两两对抗；（每周两次，每次保证全员参与）；

乒乓球团体赛：根据团队人员共分为两个队，队长分别是：李芳菲和黑淑君，首先进行两次的团队内部的淘汰赛，每个队伍选出5个人，然后进行两个队伍之间的PK，五局制；

奖项设置：

趣味奖: 四个队伍中评选出一个（每队4人）;

团体奖：二个队伍中评选出一个（每队8人）;

具体比赛规则

乒乓球趣味赛比赛项目:

一、接力托球折返跑

  1. 四个队的队员站成一排;
  2. 乒乓球放在球拍的中间，每人托运一个球;
  3. 比赛的过程中手不能触及到球;
  4. 从出发点到折返点，再从折返点返回出发点，交给另外一个人继续托运。;
  5. 如果球在比赛途中掉落在地上，选手必须捡起球重新开始比赛;
  6. 在规定的赛程内用时最少的队伍获胜;

二、吹乒乓球

  1. 每个队伍选出2~3三人，两个队伍之间进行对抗比赛，用时总和最少者为获胜方；
  2. 桌上放着十个等高的一次性纸杯，一次紧接着竖排，杯子里装满水，第一个杯子放一个乒乓球;
  3. 参赛者要将乒乓球从第一个杯子吹向第二个杯子，一个接一个地吹，乒乓球不能跳杯子，不能落到地上，直到从最后一个杯子吹出;
  4. 比赛的过程中手不能触及到球;

三、夹运乒乓球

  1. 每个队伍全员参与，两个队伍之间进行对抗比赛，用时总和最少者为获胜方；
  2. 桌子上放着5个乒乓球，选手需用筷子从甲地，运到乙地;
  3. 比赛的过程中手不能触及到球

三、趣味双打

  1. 每组派2人参加（不限男女，自由组合），要求将每位选手的左腿与同伴右腿绑在一起;
  2. 先A组与B组PK，共打5个球，输者淘汰，输方另派2人参赛，直至决出胜方;
  3. 后C组与D组PK，同样打5个球，输者淘汰，输方另派2人参赛，直至决出胜方;
  4. 先比赛的胜方与后比赛的胜方进行PK，赛制同上，直至决出胜方;

暂定4个趣味比赛项目，如有其他更有意思的玩法，欢迎推荐！

乒乓团体赛项目:

根据团队人员共分为两个队：

队伍 队长 队员
红队 菲姐 待定（8人）
蓝队 黑姐 待定（8人）

队伍内部淘汰赛：

  1. 每队内部成员进行两两对抗;
  2. 采用5局三胜制，队长自由分配队员对赛;
  3. 根据赛制，由队长挑选五位胜出者，参加团队对抗赛;

团队对抗赛:

  1. 每个队伍每次挑选出一人，迎战对方队伍;
  2. 总共进行5轮比赛，每轮5局三胜制;
  3. 5轮获胜最多的队伍为最终胜利者;
